# 采购文件填充工具

自动将 Excel 信息表中的数据填充到 Word 合同模板中。

## 功能

- ✅ 读取 Excel 信息表（【数据名】→ 数据信息）
- ✅ 自动替换 Word 模板中的【占位符】
- ✅ 将采购清单自动填入合同表格
- ✅ 保持原有格式和字体
- ✅ 支持 .doc 和 .docx 格式

## 使用方法

### 命令行版本

```bash
python main.py <信息表.xlsx> <模板.docx> <输出.docx>
```

示例：
```bash
python main.py 信息表.xlsx 采购合同模板.doc 生成的合同.docx
```

### Excel 格式要求

**信息表** 工作表：
| 数据名 | 数据信息 |
|--------|----------|
| 甲方 | 安徽中烟工业有限责任公司 |
| 乙方 | 安徽磐石办公物资有限公司 |
| 项目名称 | 办公用品采购 |

**采购物品价格** 工作表：
| 序号 | 品名 | 规格 | 单位 | 品牌 | 单价 |
|------|------|------|------|------|------|
| 1 | 中性笔 | 0.5MM | 支 | 得力 | 1.35 |

### Word 模板格式

在模板中使用 `【数据名】` 格式作为占位符，程序会自动替换为 Excel 中的对应值。

示例：
```
甲方：【甲方】
乙方：【乙方】
项目名称：【项目名称】
```

## 依赖

- Python 3.8+
- openpyxl
- python-docx

安装依赖：
```bash
pip install -r requirements.txt
```
